# Professional Portfolio - Kailash Narayan Kiradu A

An ultra-premium, modern, and highly responsive personal portfolio website designed for **Kailash Narayan Kiradu A**, a Computer Science graduate specializing in Full-Stack Java Engineering and Applied AI/ML pipelines.

## 🚀 Live Demo & Deployment
This website is optimized for static hosting platforms. It is deployed and served live on **Vercel** with full SSL protection and global CDN speed.
- Live URL: (Managed via Vercel integration)
- Custom Domain ready.

## 🎨 Design & Aesthetic Features
- **Cyber-Architect Theme**: Designed with custom HSL tokens, featuring Electric Indigo and Cyber Violet gradients on slate backgrounds.
- **Light Theme Default**: Boots up instantly in light theme and offers a toggle to switch to a persistent Dark theme cached in browser `localStorage`.
- **Zero Heavy Dependencies**: Completely decoupled from legacy frameworks, jQuery, particles.js, or heavyweight icon libraries (FontAwesome). Employs modern native browser APIs and lightweight inline SVGs.
- **Dynamic Projects Grid**: Renders projects dynamically, supporting interactive full-text search and category filtering tags.
- **Project Detail Modals**: Features responsive overlays that showcase project descriptions, tech stack pills, architecture overviews, and source code links.
- **Printable CV Mode**: Includes a tailored `@media print` style sheet, allowing visitors to print or export the website directly to a clean, professionally formatted two-page PDF resume.

## 🛠️ Verification & Development
Since the website is built entirely on native HTML5, CSS3, and ES6 Javascript, you can view and test it locally without any compilation steps:
1. Double-click `index.html` or run a local HTTP server (like VS Code's Live Server or Python's `http.server` module: `python -m http.server 8000`).
2. Open your browser and navigate to `http://localhost:8000`.
3. To build or deploy to Vercel, simply commit and push your changes to your linked Git repository; Vercel will automatically redeploy the latest changes.
